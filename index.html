Install

One file. One script tag. Press T.

1

Copy the cockpit file

Download `qa-cockpit.js` from GitHub and put it in your project folder.

2

Add it before `</body>`

<script src="qa-cockpit.js"></script>
3

Initialize it

<script>
QACockpit.init({ project: 'My Project', version: '1.0.0', devMode: true });
</script>
4

Open your build and press T

The cockpit opens over your app. Press T again or Escape to close.
